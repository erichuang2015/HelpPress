A powerful and easy-to-use knowledge base plugin for WordPress.

== Description ==

HelpPress is the knowledge base plugin for WordPress you've been looking for.

= Easy Content Organization =
Your knowledge base content can be organized by category, tag, or post format.

= Live Search Suggestions =
Suggest relevant articles to your users live as they enter search terms.

= 99% Theme Compatibility =
Plug-and-play compability with most themes. [Click here](#) to view demos.

= Developer Customizable =
Supports theme and child theme template overrides and includes dozens of actions and filters.
